Tools we built to make POS work easier

We make these small Windows utilities to simplify our own work on hospitality point-of-sale, and we share them with our customers — it seemed like a nice thing to do. Each tool is a single self-contained .exe — no installation required. Several are built with, and can be driven by, AI coding agents like Claude Code, Codex, or Gemini CLI.

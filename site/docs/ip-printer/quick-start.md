---
title: Quick start
order: 1
---

1. **Download** `MunerisIpPrinter.exe` and run it — no installer, no prerequisites.
2. Open **Settings** and add a printer; it's assigned `127.0.0.1:9100`.
3. Point your POS's printer record at that address and print.
4. The receipt appears on screen, fully rendered — text, logos, QR codes.

Add up to 15 printers, each on the next loopback address (`127.0.0.2`, `127.0.0.3`, …).

## SimCLI
`SimCli.exe` is a command-line interface for the **Oracle Simphony client**. It works against the live **WPF element tree** rather than screenshots, so automation is reliable — log in, order items, press buttons, restart the client, and more.

Because it's a predictable CLI it's naturally **agent-drivable**: Claude Code, Codex or Gemini CLI can run it directly.
